# Varausjarjestelma

Tämä työ on tehty osana Tampereen ammattikorkeakoulun C++ -ohjelmoinninperusteet kurssia. Työ on kurssin lopputyö.

Tarkoituksena on tehdä hotellin varausjärjestelmä.
Varausjärjestelmässä pystyy varaamaan huoneen hotellista, näyttämään uusimman varauksen, etsimään tehtyjä varauksia, perumaan varauksia, sekä tulostamaan kaikki varaukset.
